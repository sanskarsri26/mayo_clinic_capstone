
There are two additional dependancies for this project not included in the iOS section
- OpenCV2 : https://docs.opencv.org/4.x/d5/da3/tutorial_ios_install.html
    - Download the latest iOS pack and import the framework to the App Code/C++ folder
- Lung segmentation model : https://github.com/IlliaOvcharenko/lung-segmentation
    - Download the pth model file, there are two inside that repo and then convert it to a CoreML model
    - impoort it to the App code/Models folder
